# Panel Administrativo web con React JS

Proyecto desarrollado con React JS y Vite. Simula un panel administrativo con navegación para Clientes, Proveedor, Usuarios y Logout.

## Tecnologías

- React JS
- Vite
- React Router
- Bootstrap
- Font Awesome

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Compilación

```bash
npm run build
```

## Rutas

- `/clientes`
- `/proveedor`
- `/usuarios`
- `/logout`
